# template-backed-field-extraction-after-ocr checklist

- [ ] OCR handoff is the only bounded parsing input
- [ ] the extracted field set is explicit and reviewable
- [ ] templates or heuristics remain visible and bounded
- [ ] missing or conflicting fields remain explicit instead of being guessed away
- [ ] field values can keep source-evidence references back to the OCR handoff when needed
- [ ] locale defaults, donor parser code, and automation doctrine stay out of scope
