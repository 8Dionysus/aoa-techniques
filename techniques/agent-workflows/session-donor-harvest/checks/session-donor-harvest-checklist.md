# session-donor-harvest checklist

- [ ] the source artifact is already reviewed and bounded
- [ ] each kept record names one reusable unit rather than one topic cluster
- [ ] each kept candidate preserves at least one evidence anchor
- [ ] weak or mixed material can remain deferred or held
- [ ] the donor pack stays smaller than owner placement or promotion verdict
- [ ] the example does not reopen live execution, transcript packaging, replay, or indexing
- [ ] derivative routing or graph surfaces do not become first-authoring targets
- [ ] the public wording stays reusable and sanitized
