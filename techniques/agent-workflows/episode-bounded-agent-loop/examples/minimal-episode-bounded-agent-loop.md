# minimal episode-bounded agent loop

Episode goal:

- draft one bounded seed bundle and record the remaining overlap question

Checkpoint artifact:

```markdown
# Episode Checkpoint
- completed: drafted the seed bundle skeleton
- blocker: overlap with `AOA-T-0001` still needs boundary review
- next_decision: `escalate`
```

Why this example stays bounded:

- the episode ends at one visible checkpoint instead of running indefinitely
- the next move is explicit: continue, stop, or escalate
- it does not import budgets, task trackers, launchd supervision, or a full autonomous mission runtime
