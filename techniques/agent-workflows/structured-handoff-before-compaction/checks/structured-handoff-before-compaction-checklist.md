# structured-handoff-before-compaction checklist

- [ ] a real compaction, rollover, or episode boundary is named explicitly
- [ ] the handoff artifact is written before context loss rather than after it
- [ ] completed work, in-progress or blocked work, and the next step are explicit
- [ ] the packet includes concrete continuation references such as files, commits, or artifacts
- [ ] the example stays smaller than transcript packaging, witness export, or mailbox transport
- [ ] the technique does not rely on hidden memory, launchd behavior, or one orchestrator runtime
- [ ] the bundle stays smaller than the live `phase-synchronized-agent-handoff` narrowing lane
- [ ] the public wording remains reusable and sanitized
