# Minimal Receipt Review

1. Collect the source-owned receipts for one repeated degraded hybrid-query family.
2. Group them by `surface=hybrid-query` and one named `stressor_class`.
3. Record facts from the receipts and linked run artifacts.
4. Write hypotheses separately from facts.
5. Propose one narrow change, such as clearer fallback labeling or stronger evidence linkage.
6. Name the next receipt or eval surface that should show improvement after the change.
