# Upstream Mirroring With Provenance Checklist

- Name each mirrored item in one explicit source manifest before syncing it locally.
- Keep one local destination per mirrored upstream source.
- Preserve adjacent provenance or attribution metadata for each mirrored copy.
- Make upstream ownership readable after sync.
- Keep mirrored payloads separate from local wrapper or curation notes.
- Re-run sync through the declared manifest path instead of defaulting to ad hoc local patching.
- Review both mirrored content and provenance metadata after refresh.
