# lightweight-snapshot-checklist

- `README` stays a short human-facing entrypoint
- `MANIFEST`, when present, stays a compact machine or mixed snapshot
- long run IDs and historical metric blocks are absent from snapshot docs
- detailed execution state lives in canonical docs such as `TODO`, `PLANS`, `RUNBOOK`, `DECISIONS`, or `SESSION_*`
- snapshot docs link outward instead of copying canonical detail
- current-state evidence points to CI or the latest session-style snapshot
- snapshot docs remain readable in one screen or close to it
