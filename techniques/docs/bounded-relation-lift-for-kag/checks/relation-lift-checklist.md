# Relation Lift Checklist

Use this checklist to confirm that direct relations still behave like bounded adjacency hints.

- Each relation points to a real direct target bundle.
- Current consumers use the edge for one-step inspection or navigation only.
- The relation vocabulary stays small and human-reviewable.
- Reviewers can still answer "why does this edge exist?" from markdown or semantic review surfaces.
- The workflow does not require weighting, multi-hop traversal, or a new graph/export layer.
