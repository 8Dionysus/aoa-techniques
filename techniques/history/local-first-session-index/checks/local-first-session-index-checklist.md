# local-first-session-index checklist

- [ ] source session artifacts already exist before indexing starts
- [ ] index entries keep stable references back to the source artifact
- [ ] the index can be rebuilt from local sources without hidden memory or recall state
- [ ] capture behavior stays outside the technique
- [ ] transcript packaging or export stays outside the technique
- [ ] hosted dashboards, analytics, and remote sync stay outside the technique
- [ ] query or browse results remain derivative and do not replace source authority
- [ ] the example uses public-safe metadata and paths only
