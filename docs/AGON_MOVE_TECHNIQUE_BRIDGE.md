# Agon Move Technique Bridge

## Purpose

Wave IV gives `aoa-techniques` a first candidate intake surface for Agon lawful moves.

These are not promoted techniques. They are requests for future reusable practice work behind center-owned lawful moves.

## Boundary

`aoa-techniques` may later own:

- reusable practice behind a move;
- adaptation notes;
- practice validation methods;
- portability guidance.

It must not own:

- lawful move vocabulary;
- arena session law;
- skill workflow execution;
- proof verdicts;
- durable scars;
- routing authority;
- ToS canonization.

## Candidate index

The seed adds:

```text
config/agon_technique_binding_candidates.seed.json
generated/agon_technique_binding_candidates.min.json
```

Each candidate is `requested_not_landed`.

Promotion into a real technique requires the normal `aoa-techniques` authoring and review path.

## One-line rule

The center names the legal word. `aoa-techniques` may later teach the hand how to use it.
